Common CSS properties

by Miranda

CSS Property Description

font-family

 typeface of an element

font-size

 size of text displayed

font-weight

 how thin or thick text is displayed

text-align

 places text in the left, right, or center of its parent container

color

 color of the text

background-color

 color behind the text

opacity

 how not transparent the text is

background-image

 set the background of an element to an image
